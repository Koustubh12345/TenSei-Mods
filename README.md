### Instructions:
- Download and install the `TenSei-Mods.apk`.
- Open the app and grant **Root Access** (Magisk or KernelSU).
- Select your desired modifications from the menu.
- Click **"Apply Changes"** to restart SystemUI and finalize settings.

### READ:
- **Root is mandatory.** This app will not function on non-rooted devices.
- **Integrity Lock:** This app includes a deep-link security scan. Unauthorized modification of the assets will trigger a tamper-protection lock.
- **Disclaimer:** The developer is not responsible for any system instability or bootloops. Use at your own risk.
