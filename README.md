# 🌌 TenSei Mods (v1.0)

**TenSei Mods** is an advanced, root-only customization suite specifically engineered for **Oplus, ColorOS, OxygenOS, and RealmeUI** devices and ports. It combines a high-performance Smali/Java bridge with a premium "Atmospheric" glass-morphism UI to provide deep system-level modifications without the need for manual framework editing.

---

## ⚡ Key Features

* **Native Root Integration:** Directly interacts with system properties and global settings via a secure root bridge.
* **Persistent Configuration:** All modifications are stored within the APK's internal database (`SharedPreferences`), ensuring your setup survives app restarts.
* **Integrity Protection:** Features a deep-link security scan to prevent unauthorized tampering or modification of the source.
* **Oplus Optimized:** Specifically tailored for Oplus-based ROMs and high-end device ports.

## 📥 Installation & Usage
1.  **Requirements:** Android 11+ and **Root Access** (Magisk or KernelSU).
2.  Install the `TenSei-Mods.apk`.
3.  Grant Root permission when prompted.
4.  Navigate through the categories and apply your desired mods.
5.  Click **"Apply Changes"** to restart SystemUI and finalize the setup.
