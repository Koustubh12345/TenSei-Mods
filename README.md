# 🌌 TenSei Mods (v1.0)

**TenSei Mods** is an advanced, root-only customization suite specifically engineered for **Oplus, ColorOS, OxygenOS, and RealmeUI** devices and ports. It combines high-performance Smali logic with a premium "Atmospheric" glass-morphism UI to provide deep system-level modifications.

### ⚡ Features
- **Native Root Integration:** Direct interaction with system properties and global settings via a secure root bridge.
- **Persistent Configuration:** All modifications are stored in the APK's internal database (`SharedPreferences`), surviving app restarts.
- **Integrity Protection:** Features a deep-link security scan to prevent unauthorized tampering of the source assets.
- **Gaming Spoof:** Unlocks 90/120 FPS in BGMI/CODM and other supported games via high-end device profiles.
- **Charging Mod:** Dynamic battery colors (Green/Blue/Purple) and custom VOOC wattage display.

### 🚀 Instructions
1. Download and install the `TenSei-Mods.apk`.
2. Open the app and grant **Root Access** (Magisk or KernelSU) when prompted.
3. Select your desired modifications from the **Misc** or **Statusbar** sections.
4. Click **"Apply Changes"** to restart SystemUI and finalize your setup.

### ⚠️ READ BEFORE USE
- **Root is Mandatory:** This app interacts with protected system properties and will not function without root permissions.
- **Integrity Lock:** This app includes a tamper-detection system. Unauthorized modification of the APK's internal assets will trigger a lock screen.
- **Disclaimer:** The developer is not responsible for any system instability, bootloops, or data loss. Use these modifications at your own risk.
